Crystal Collector!

use logic and addition to beat the computer

Game Rules

  • Goal: Collect the correct number of crystals.
  • Collecting Crystals: Click on each crystal to collect them.
  • Scoring: Each of the four(4) crystals below is worth 1-12 points. The value is random, changes with each game, and is hidden unless clicked.
  • Winning: Match the number of crystals you collect to the number in the amythest crystal cluster.
  • Careful! If you collect too many crystals you lose the game!
